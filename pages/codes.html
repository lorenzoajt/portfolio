  • Greekinator: a Text Based approach to generate melodies using artificial intelligence.

    Greekinator uses the intuitive power of Deep Learning and Natural Language Processing to create musical ideas.

    It uses sentiment analysis to classify the text and, given the sentiment, it assigns a greek mode(you can hear what I mean here at min 12:52). Then it creates a melody following the chord progression using Magenta's pretrained models.
